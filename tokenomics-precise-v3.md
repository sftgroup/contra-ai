## 0. 核心公理

**价格稳步上涨的充要条件**：

```
dP/dt > 0  ⇔  d(Demand)/dt > d(Supply)/dt
```

在任意时间点，需求增量必须大于供给增量。

## 3. 流动性池建立后的价格动力学

```

### 3.2 无防护的最坏模拟

假设 35% 用户卖出手中 50% 的零撸代币：

```

**价格：$0.00995 → $0.00817（仅跌 -18%）✅ 可接受**
