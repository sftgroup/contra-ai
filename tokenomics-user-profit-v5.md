## Part 1: 20 天零撸期的精确模拟

### 1.1 核心假设

| 参数 | 数值 | 逻辑 |
|------|------|------|
| 第 1 天上线注册用户 | 5,000 | 种子用户 + 内测社群 |
| 日均自然增长 | +3,000 | 社群推广 + 130 国预热 |
| 20 天后活跃用户（DAU） | 62,000 | 累计注册约 8 万，DAU 占比 77.5% |
| 每日签到基础领取 | 200 CAI | 轻量低门槛 |
| KYC + 钱包绑定 | +300 CAI | 身份验证奖励 |

| 邀请奖励 | 1,000 CAI/人 | 被邀者完成 KYC 后发放 |
| KYC 完成率 | 60% | 防刷 + 合规 |
| 任务完成率 | 50% | 保守估计 |
| 平均邀请成功率 | 每个活跃用户邀请 0.3 人/天 | 前 5 天裂变慢，后 15 天加速 |

### 1.2 每日释放量模型

零撸池总量 37.5 亿，其中直接零撸 22.5 亿。这里的设计逻辑：**20 天很短，不需要过度克制释放**。释放主要集中在前期以建立用户基数。

```
Day 1:  5,000 用户 × (200 + 300×0.6 + 500×0.5) = 5,000 × 630 = 3,150,000 CAI
Day 5:  17,000 用户 × 630 = 10,710,000 CAI  
Day 10: 32,000 用户 × 630 = 20,160,000 CAI
Day 15: 47,000 用户 × 630 = 29,610,000 CAI
Day 20: 62,000 用户 × 630 = 39,060,000 CAI
```

**20 天直接零撸总计（含邀请）：**

| 阶段 | 天数 | 日均用户 | 日均释放(CAI) | 阶段总释放 |
|------|------|----------|---------------|------------|
| 冷启动期 | Day 1-5 | 5,000→17,000 | 3.15M→10.71M | 34,650,000 |
| 增长期 | Day 6-10 | 17,000→32,000 | 10.71M→20.16M | 77,175,000 |
| 加速期 | Day 11-15 | 32,000→47,000 | 20.16M→29.61M | 124,425,000 |
| 爆发期 | Day 16-20 | 47,000→62,000 | 29.61M→39.06M | 171,675,000 |
| **20 天总计** | | | | **407,925,000** |

邀请裂变释放量（20 天内累计）：

```
累计被邀请完成 KYC 人数 ≈ 8 万总注册 × 0.6 × 0.3 × 20 × 0.5（重复去重）= 144,000 人
邀请奖励释放 = 144,000 × 1,000 = 144,000,000 CAI
```

**20 天零撸 + 邀请总释放 ≈ 4.08 亿 + 1.44 亿 = 5.52 亿 CAI**

占零撸池（37.5 亿）的 **14.7%**，占总量（75 亿）的 **7.36%**。

> 📊 **结论：20 天释放仅占总量的 7.36%，流通量极度稀缺。这是后面价格上涨的核心基础。**

### 1.3 单个用户 20 天收益

**忠实用户画像（每日签到 + KYC + 全部基础任务 + 活跃邀请）：**

```
每日基础 = 200 (签到) + 300 (KYC, 一次性) + 500 (任务) = 第1天 1,000, 之后每天 700
20 天签到+任务 = 1,000 + 700 × 19 = 14,300 CAI
邀请奖励 = 每天邀请 0.3 人成功 × 20 天 × 1,000 CAI = 6,000 CAI
────────────────────────────────────
忠实用户 20 天总量 ≈ 20,300 CAI
```

**纯撸毛用户（仅签到，无 KYC，无任务，无邀请）：**

```
每日仅签到 = 200 CAI
20 天总量 = 200 × 20 = 4,000 CAI
```

> 📊 **差距：忠实用户 = 20,300 CAI vs 纯撸用户 = 4,000 CAI → 5 倍差距。忠实用户值得忠诚。**

|------|------|-----------|-----------|-----------------|
| 验证期 | Day 1-3 | 0→1 | 3 | 15,000 |

| 爆发期 | Day 8-14 | 5→7 | 64 | 320,000 |
| 冲刺期 | Day 15-20 | 7→6 | **100** | **500,000** |

每个专家 10,000 USDC，其中 **5,000 USDC 进入合约挂单买入池**。

20 天内合约累计买入力 = 100 × 5,000 = **500,000 USDC**

### 2.2 挂单市场供需模型

挂单市场是**自由定价订单簿**，合约每天用专家资金从低到高扫单。

**用户挂单心理：**
- 纯撸用户：撸到即卖，"有利润就行"，挂低价
- 持有者：看到价格被推高，不想卖，挂高价试探
- 投机者：看到专家持续买入，抢低价单再挂高价

### 2.3 第 20 天关键数据

```
20 天合约累计买入 ≈ 500,000 USDC
20 天累积成交 CAI 总量 ≈ 8,000,000 CAI（平均每日 40 万 CAI）
最后成交价 = $0.01000
加权平均成交价 ≈ $0.00310

这个价格下：
- 忠实用户 20,300 CAI 的挂单价值 = 20,300 × $0.010 = $203.00
- 纯撸用户 4,000 CAI 的挂单价值 = 4,000 × $0.010 = $40.00
```

> 🎯 **价格从 $0.00005 涨到 $0.01，20 天涨幅 200 倍。不是靠炒作，是靠专家资金的持续买入 + 流通量极度稀缺。**

### 2.4 为什么 $0.01 是合理的？

```
第 20 天流通市值 = 5.52 亿 × $0.01 = $552 万
FDV（完全稀释估值） = 75 亿 × $0.01 = $7,500 万

对于一个 130 国部署 + 100 个付费专家（每人投了 $10,000）+ 
即将建立 50 万 USDC 流动池的 AI 平台 → $7,500 万 FDV 极度保守。
类似阶段的项目 FDV 通常在 $5,000 万 - $2 亿。
```

## Part 4: 什么驱动价格上涨？

### 4.1 五大驱动力模型

**力 1: Staking 锁仓（减少流通量）**

| 时间 | Staking 比例 | 锁仓 CAI | 流通 CAI | 流通占总量 |
|------|-------------|----------|----------|-----------|
| M0 | 15% | 0.83 亿 | 5.52 亿 | 7.36% |
| M6 | 35% | 4.0 亿 | 13 亿 | 17.3% |
| M12 | 45% | 9.2 亿 | 18 亿 | 24.0% |
| M18 | 50% | 15 亿 | 22 亿 | 29.3% |
| M24 | 55% | 21 亿 | 26 亿 | 34.7% |
| M36 | 60% | 34 亿 | 28 亿 | 37.3% |

> 💡 Staking 年化 12-18%，吸引超过 50% 的代币锁仓。流通量增长远慢于锁仓增长。

**力 2: 专家铸造带来的买盘**

每个专家铸造 = 5,000 USDC 进入生态（直接或间接买入 CAI）：

| 时期 | 专家累计 | 累计 USDC 买盘 |
|------|---------|---------------|
| M0 (20 天) | 100 | 50 万 |
| M3 | 500 | 250 万 |
| M6 | 1,200 | 600 万 |
| M12 | 3,000 | 1,500 万 |
| M18 | 5,000 | 2,500 万 |
| M24 | 8,000 | 4,000 万 |
| M36 | 12,000 | **6,000 万** |

> 💡 这 6,000 万 USDC 是**真实买盘**，不是投机资金。随着平台服务质量提升，专家收入增加 → 更多专家加入 → 更多买盘 → 正反馈循环。

**力 3: 效用需求（咨询/AI Agent/节点）**

| 效用类型 | 需求 CAI 量 | 作用 | 对价格影响 |
|----------|------------|------|-----------|
| 咨询支付 | 用户用 CAI 支付咨询费 | 消耗 + 需求 | 减少流通 |
| AI Agent 订阅 | 月费 500-2000 CAI | 持续购买 | 稳定需求 |
| 节点质押 | 每节点 10,000-50,000 CAI | 大额锁仓 | 显著减少供应 |
| Agent 创作者手续费 | 每次交易燃烧 0.1-1% | 通缩 | 总量减少 |

```
月化效用需求估算：
M6:  200 万 CAI/月 → $12 万需求
M12: 500 万 CAI/月 → $90 万需求
M24: 1,200 万 CAI/月 → $1,020 万需求
M36: 3,000 万 CAI/月 → $7,500 万需求
```

**力 4: 销毁/通缩**

```
交易手续费燃烧：每笔 DEX 交易燃烧 0.1%
Agent 市场交易燃烧：每笔燃烧 0.5%
节点惩罚燃烧：违规节点质押被销毁

年化燃烧量估算：
Year 1: ~500 万 CAI（交易量增长期）
Year 2: ~2,000 万 CAI（生态活跃期）
Year 3: ~8,000 万 CAI（成熟期）

3 年累计销毁 ≈ 1.05 亿 CAI，占总量的 1.4%
```

**力 5: 机构与团队释放（12 个月后才开始）**

```
机构融资（22.5 亿 CAI）：
- M12 开始线性释放，24 个月释放完毕
- 每月释放 = 22.5 亿 / 24 = 9,375 万 CAI/月

创始团队（15 亿 CAI）：
- 每 2 万注册释放 1%
- 预计 M6 达到 200 万注册，团队全部释放完毕
- 但：团队 tokens 受限于 staking 协议，释放后需锁仓 6 个月
```

> ⚠️ **关键：机构在 M12 才有释放，在此之前只有零撸流通。12 个月的保护期让价格有足够时间建立支撑。**

### 4.2 月化合理增长率推导

**从供需方程出发：**

```
月度新需求（USDC） = 专家买入(5K×新增数) + 效用需求(咨询/Agent/节点) + 投机需求(新用户)
月度新增流通(CAI) = 零撸释放 + 邀请释放 + (M12后)机构释放 + 团队释放 - 锁仓增长 - 燃烧

目标：需求增速 > 供应增速，价增长 15-25%/月
```

| 月份 | 新增需求(USD) | 新增流通(USD@现价) | 供需比 | 价格乘数 |
|------|-------------|------------------|--------|---------|
| M1-3 | 150 万 | 50 万 | 3.0× | 1.80× |
| M4-6 | 300 万 | 120 万 | 2.5× | 2.40× |
| M7-12 | 800 万 | 400 万 | 2.0× | 3.00× |
| M13-18 | 1,500 万 | 900 万 | 1.7× | 2.33× |
| M19-24 | 2,500 万 | 1,500 万 | 1.7× | 2.02× |
| M25-36 | 4,000 万 | 2,800 万 | 1.4× | 2.14×（但基数大，增速放缓到 9-12%） |

> 🎯 **供需比始终 > 1，意味着任何时候卖盘都被买盘吸收。这是模型稳健性的核心证明。**

### 5.3 池子建立后如果价格跌了怎么办？

**这是用户心理的核心问题。我们的设计：**

**短期缓冲机制（防止恐慌抛售）：**

1. **Staking 高收益窗口**
   - 第 20-30 天：Staking 年化 50%（仅此 10 天窗口）
   - 锁定代币 30 天 → 大量流通被锁，缓冲抛压
   - 锁仓奖励：额外 CAI + 平台积分

2. **挂单市场平滑过渡**
   - 第 20 天不是"瞬间切换"，挂单市场保留 48 小时
   - 用户在挂单市场和 DEX 之间套利 → 价格自然收敛

3. **信息发布节奏**
   - Day 20: 公布第一批 100 专家名单和画像 → 增强持有信心
   - Day 22: 发布 AI Agent 预览 → 展示效用价值
   - Day 25: 公布节点经济白皮书 → Staking 需求预期

4. **如果跌了 20-30%（概率低但预案）**
   - 触发专家铸造加速计划：新专家享受 8,000 USDC 优惠
   - 团队动用生态激励池（11.25 亿）中的 1% 做市场操作
   - 发布路线图加速公告
   - **关键信息给用户："跌了是加仓机会，不是逃跑信号"**

### 模型的鲁棒性

这个模型的核心假设是：
1. **月化增长 12-25%**（不是 meme coin 的不可持续增长）
2. **专家持续加入**（平台价值驱动，不是投机）
3. **Staking 锁仓 > 50%**（用户在赚钱，自然锁仓）
4. **真实效用需求**（咨询、Agent、节点是真金白银的使用场景）

如果任何一个假设不成立，模型仍有余量：
- 如果专家增速慢 30% → 价格增速降至 10-15%/月 → M36 价格约 $1.20-$1.80（仍可观）
- 如果 Staking 比例仅 30% → 流通稍多 → M36 价格约 $1.50-$2.00
- 即使最差情况（专家停止增长、用户大量抛售）→ 零撸释放仅 7.36% 总量，池子有 50 万 USDC 支撑 → 价格底线约 $0.002-$0.005
