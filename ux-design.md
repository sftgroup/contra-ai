## 2. 关键用户流程

### 2.1 铸造 NFT 主流程

```
首页 Hero
 │
 ├──→ 点击 "铸造合伙人NFT"
 │      │
 │      ▼
 │    铸造页 (/mint)
 │      │
 │      ├── [未连接钱包] → 点击 → Web3Modal 弹窗
 │      │      ├── MetaMask
 │      │      ├── WalletConnect (扫码)
 │      │      ├── Coinbase Wallet
 │      │      └── Trust Wallet
 │      │
 │      ▼ [已连接]
 │    选择链卡片
 │      ├── Base (已铸造: X/500)
 │      ├── BSC   (已铸造: X/500)
 │      ├── ETH   (已铸造: X/500)
 │      └── Solana(已铸造: X/500)
 │      │
 │      ▼ 点击某链
 │    铸造确认弹窗
 │      ├── 链名称 + Logo
 │      ├── 费用: 10,000 USDC
 │      ├── Gas 预估: ~$X.XX
 │      ├── 你的 NFT 等级: [黄金/白银/青铜]
 │      └── [确认铸造] [取消]
 │      │
 │      ▼ 确认
 │    交易处理中...
 │      ├── Loading 动画
 │      └── "请在钱包中确认交易"
 │      │
 │      ▼ 链上确认
 │    铸造成功!
 │      ├── 🎉 烟花/粒子动效
 │      ├── NFT 卡片翻转展示
 │      ├── Token ID + 铸造链 + 时间
 │      ├── [进入 Dashboard] CTA
 │      └── [分享到 Twitter]
 │
 └──→ 点击 "阅读白皮书" → 白皮书内嵌页
```

## 3. 交互规范

### 3.1 动效规范

| 场景 | 动效 | 时长 |
|---|---|---|
| 页面切换 | Fade + Slide Up | 300ms |
| 数据数字变化 | Count Up 动画 | 800ms |
| 卡片 Hover | Scale(1.02) + Glow Border | 200ms |
| 按钮 Hover | Scale(1.05) + 亮度提升 | 150ms |
| Modal 出现 | Scale(0.95→1) + Fade | 250ms |
| 铸造成功 | Particle Fireworks + Card Flip | 2000ms |
| Tab 切换 | 下划线 Sliding | 250ms |
| 进度条更新 | Smooth Transition | 500ms |
| 语言切换 | Fade Out → Fade In | 200ms |

### 3.2 微交互

| 元素 | 交互 |
|---|---|
| 导航链接 | Hover 下划线从左滑入 |
| 链选择卡片 | 选中态：边框发光 + 内部勾选标记 |
| 铸造按钮 | 脉冲呼吸动效（吸引点击） |
| 复制按钮 | 点击变"已复制" + check 动画 |
| 数据看板 | 数字增长动画（仅首次加载） |
| 售罄标记 | 灰度 + "已售罄" 红色标签 |
| 未开放 Tab | 置灰 + Hover "即将开放" Tooltip |

### 3.3 反馈机制

| 场景 | 反馈 |
|---|---|
| 连接钱包成功 | Toast：绿色 "钱包已连接" |
| 交易发送 | Toast：蓝色 "请在钱包中确认" |
| 交易确认中 | 按钮 Loading + 进度提示 |
| 交易成功 | 全屏成功动效 + 信息卡片 |
| 交易失败 | Toast：红色 "铸造失败，请重试" |
| 复制成功 | 按钮文字变 "已复制!" 1.5s |
| 网络错误 | Toast：黄色 "网络异常，正在重试..." |

## 4. 响应式策略

### 4.1 断点

| 断点 | 宽度 | 布局 |
|---|---|---|
| Mobile | 320-767px | 单列，堆叠 |
| Tablet | 768-1023px | 双列 |
| Desktop | 1024px+ | 多列，侧边栏 |

### 4.2 关键适配

| 组件 | Mobile | Desktop |
|---|---|---|
| 导航 | 汉堡菜单 | 水平导航栏 |
| 语言切换 | 底部 Sheet | 下拉菜单 |
| Hero | 单列堆叠 | 左右分栏 |
| 白皮书 | 顶部章节下拉 | 左侧固定导航 |
| 链选择 | 2×2 网格 | 4 列横排 |
| Dashboard | 单列 | 双列布局 |
| 里程碑 | 简单列表 | 交互时间线 |
| 钱包按钮 | 图标模式 | 图标+文字 |

### 4.3 移动端特殊处理

- 钱包连接弹窗全屏（避免小弹窗难操作）
- 铸造确认使用底部 Sheet（拇指友好）
- NFT 数据看板缩小为卡片轮播
- 表格数据改为卡片列表

## 5. 可访问性 (A11Y)

| 元素 | 措施 |
|---|---|
| 配色 | 不纯靠颜色区分 NFT 等级（+ 图标 + 文字） |
| 对比度 | 所有文字 ≥ WCAG AA (4.5:1) |
| 键盘 | Tab 键可导航所有交互元素 |
| 焦点 | 可见 Focus Ring |
| 表单 | aria-label / aria-describedby |
| 图标 | svg aria-hidden + 文字替代 |
| 动效 | 支持 prefers-reduced-motion |
| 语义 | heading 层级正确 (h1→h2→h3) |

## 6. 空态与边界

| 状态 | 处理 |
|---|---|
| 未连接钱包 | 铸造按钮置灰 + "请先连接钱包" 提示 |
| 已连接但无该链 USDC | 显示余额 + "余额不足" 警告 |
| 某链已售罄 | 对应卡片灰显 + "已售罄"标签 |
| 全部售罄 (500/500) | 整页切换售罄状态 + "感谢支持" |
| Dashboard 未铸造 | 空态引导卡片 |
| Dashboard 已铸造 | 完整合伙人面板 |
| 网络切换 | 提示"请切换到支持的链" |
| 加载中 | Skeleton 占位 |
| 404 | 自定义科技感 404 页 |

## 7. 视觉设计方向建议

### 7.1 色彩体系

| 用途 | 色值 |
|---|---|
| 主背景 | #0A0E27 (深蓝黑) |
| 卡片背景 | #111633 / #1A1F45 |
| 主强调色 | #3B82F6 (科技蓝) |
| 辅助强调 | #8B5CF6 (紫色) |
| 金色渐变 | #F59E0B → #D97706 (黄金NFT) |
| 银色 | #9CA3AF (白银NFT) |
| 铜色 | #B45309 (青铜NFT) |
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 错误 | #EF4444 |
| 文字主 | #F1F5F9 |
| 文字次 | #94A3B8 |

### 7.2 字体

| 用途 | 字体 |
|---|---|
| 标题 | Inter / SF Pro Display (无衬线) |
| 正文 | Inter / Noto Sans (多语言支持) |
| 阿拉伯语 | Noto Naskh Arabic |
| 泰语 | Noto Sans Thai |
| 韩语 | Noto Sans KR |
| 日语 | Noto Sans JP |
| 中文 | Noto Sans TC |
| 代码/Mono | JetBrains Mono |

### 7.3 关键视觉元素

- **背景**：暗色渐变 + 动态粒子网格线
- **卡片**：半透明玻璃态 (Glassmorphism) + 发光边框
- **按钮**：渐变背景 + 微光扫过动效
- **图标**：线性风格 + 渐变填充
- **数据数字**：大号粗体 + 渐变文字 + Count Up
- **NFT 等级**：金属质感卡片（金/银/铜材质感）
- **区块链元素**：Hex 网格、节点连线、数据流动线
