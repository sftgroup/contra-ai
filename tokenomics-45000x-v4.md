## 3. 专家铸造的通缩正反馈

$5,000 USDC 买入力随价格上涨急剧缩水：

| 价格 | 5,000 U 能买 |
|------|------------|
| $0.01 | 500,000 CAI |
| $0.10 | 50,000 CAI |
| $1 | 5,000 CAI |
| $10 | 500 CAI |
| $100 | 50 CAI |
| $450 | **11 CAI** |

**价格越高 → 买入量越少 → 流通增量越小 → 价格拉升越容易**
