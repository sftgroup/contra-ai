## 一、项目背景

### 平台定位
**中国AI算力 · 服务全球 · 收益回流 · 普惠民生**

依托阿里云AI算力出海，搭建覆盖全球130国的AI专家咨询平台。全球用户通过本地化方式支付服务费用。

### 核心价值
1. 中国AI算力，为全球提供智能专家服务
2. 全球服务收益，100%合规法币结算回归

### 运行逻辑
1. **算力底座**：深度合作阿里云，使用低成本、高稳定的国产AI算力
2. **全球平台**：各国AI专家入驻，为全球用户提供合规的智能咨询服务
3. **130国数据本地驯化**：让AI代替人类咨询，完全去中心化服务
4. **DAO治理一键仲裁**：知识不再被白嫖

## 二、官网功能需求

### 2.1 官网首页
- 科技感 + 现代化设计风格
- 展示平台核心信息：
  - 白皮书入口
  - 里程碑（技术里程碑 + 运营里程碑）
  - 社交媒体链接（Twitter、YouTube、Discord等）
  - GitHub 代码库（未来开放）
- **钱包登录入口**（Web3Modal / WalletConnect）

### 2.2 合伙人NFT铸造页面
- 铸造费用：**10,000 USDC** / 枚
- 支持多链：**Base / BSC / ETH / Solana**
- **总量 500 个**（所有链合计）
- 实时展示：
  - 每条链已铸造数量
  - 所有链总铸造数量
  - 剩余可铸造数量
- 铸造流程：连接钱包 → 选择链 → 确认支付 → 铸造成功 → 展示NFT

### 2.3 用户 Dashboard
登录后进入个人面板，分 Tab 展示：

| Tab | 身份 | 当前状态 |
|---|---|---|
| 用户 | 基础注册用户 | 待开发 |
| **合伙人** | NFT铸造用户 | **✅ 已开放铸造** |
| 专家 | 入驻顾问/战略官 | 待开发 |
| KOL | 意见领袖 | 待开发 |

**合伙人 Dashboard** 需展示：
- NFT 资产信息（链上地址、Token ID、铸造链）
- 分红权益说明
- 平台流水数据（若有）
- 推荐链接（裂变机制）
- 当前股东权益清单

## 三、股东权益体系

### 3.1 核心权益
1. **永久收益**：链上铸造平台股东NFT，永久享受平台流水分佣（500股东占平台利润20%）
2. **Token分配**：20% $CAI代币，500人免费分配（每2万人入驻释放1%）
3. **上市权益**：新加坡法定公司若美国上市成功，分20%股份（500人平均分）

### 3.2 阶梯奖励
| 批次 | 人数 | 分佣加权 | NFT等级 | 额外权益 |
|---|---|---|---|---|
| 第1批 | 前100人 | 10% | 黄金创始股东 | 官网荣誉墙 |
| 第2批 | 101-300人 | 5% | 白银创始股东 | 官网荣誉墙 |
| 第3批 | 301-500人 | 3% | 青铜创始股东 | 官网荣誉墙 |

### 3.3 裂变机制
- 推荐1人：推荐人额外获得被推荐人平台流水分佣加权（长期叠加）
- 分享4人各投1万 → 赚1万佣金 + 免费成为股东
- 自己投1万 + 分享3人 → 退回本金 + 免费成为股东

## 五、未来商业模式（需在官网体现）

### A. Agent 市场
- 专家可设置 AI Agent
- Agent 在授权范围内自主工作
- 在市场中被雇佣，7×24小时创造收入
- Agent 收入归专家所有

### B. 节点经济
- 用户对话需支付代币
- 节点提供算力获得代币收入
- 类似 "Token 中转站" 收入模式
- 节点 = 算力贡献者
